This project demonstrates an end-to-end **Data Analytics workflow** using industry-standard tools. ## Dataset

* **Type:** Customer / Shopping behavior dataset
* **Format:** CSV / Excel
* **Contents:** Customer details, purchase behavior, product categories, spending patterns
* **Size:** Medium-scale dataset suitable for analysis and visualization

## Tools & Technologies

* **Python:** Data loading, cleaning, and Exploratory Data Analysis (EDA)
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn
* **SQL:** MySQL Workbench (data storage and querying)
* **BI Tool:** Power BI (interactive dashboard)
* **Reporting:** MS Word / PDF
* **Presentation:** Gamma (AI-based PPT creation)

## Project Workflow / Steps

### 1. Data Loading (Python)

* Imported raw dataset using Pandas
* Verified schema, data types, and structure

### 2. Exploratory Data Analysis (EDA / PDA)

* Summary statistics (mean, median, mode, std)
* Distribution analysis
* Outlier and missing value detection

### 3. Data Cleaning

* Handled missing values
* Removed duplicates
* Renamed and created new columns
* Dropped irrelevant fields

### 4. Feature Engineering

* Created derived columns for analysis
* Prepared data for modeling and reporting

### 5. SQL Operations (MySQL Workbench)

* Created database and tables
* Inserted cleaned data
* Executed SQL queries:

  * SELECT, WHERE, GROUP BY
  * ORDER BY, HAVING
  * Aggregate functions (SUM, AVG, COUNT)

### 6. Model & Score Checking (Basic Analytics)

* Applied simple analytical models (e.g., Linear Regression)
* Evaluated results using model score

### 7. Power BI Dashboard

* Connected SQL / cleaned dataset to Power BI
* Built interactive visuals:

  * Sales trends
  * Customer behavior insights
  * Category-wise performance
* Used slicers and filters for interactivity

### 8. Reporting

* Created a structured project report
* Included problem statement, methodology, insights, and conclusions

### 9. Presentation (Gamma)

* Converted project insights into a professional PPT
* Used visuals, charts, and key findings for stakeholder presentation

## How to Run the Project

1. Clone or download the project files
2. Open Python scripts / Jupyter Notebook
3. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn
   ```
4. Run Python scripts for cleaning and EDA
5. Load cleaned data into MySQL using SQL Workbench
6. Execute SQL queries for analysis
7. Open Power BI file and refresh data
8. Review final report and presentation
